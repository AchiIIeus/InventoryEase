# InventoryEase

InventoryEase was a Flask-based inventory management web app that helped small store owners track stock and basic sales with an easy dashboard.

## Features
- User login and registration
- Inventory CRUD (create, read, update, delete)
- Total stock value and low-stock report
- Simple search

## Getting Started
```bash
git clone <your-repo-url>.git
cd InventoryEase
python -m venv venv
# Windows
venv\Scripts\activate
# Mac/Linux
source venv/bin/activate
pip install -r requirements.txt
python app.py
